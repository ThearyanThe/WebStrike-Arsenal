# WebStrike-Arsenal

A collection of advanced, custom‑written exploit modules and penetration testing tools for modern web vulnerabilities. Designed for red team operations, bug bounty hunting, and security research.

## Features
- Modular Python framework with core libraries for HTTP requests, encoding, and logging.
- Coverage of major vulnerability classes: SQLi, XSS, SSTI, Deserialization, SSRF, XXE, JWT, File Upload, LFI/RFI, HTTP Smuggling, CSRF, CORS, GraphQL, NoSQL, API security, WebSockets, CI/CD attacks, and post‑exploitation.
- Clean, extensible architecture following Python best practices.

## Installation
```bash
git clone https://github.com/your-username/WebStrike-Arsenal.git
cd WebStrike-Arsenal
pip install -r requirements.txt
```

## Usage
```bash
python webstrike.py --help
```
